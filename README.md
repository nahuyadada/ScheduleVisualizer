# ScheduleVisualizer

A simple static web app to visualize schedules.

## Getting Started

1. Open `index.html` in your browser (double-click or drag into a tab).
2. Ensure `styles.css` and `app.js` are in the same folder.

## Project Structure

- `index.html` — main page
- `styles.css` — styling
- `app.js` — interactive logic

## Development Notes

No build step required. Edit the files and refresh the browser.
